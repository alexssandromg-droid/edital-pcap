# 👮‍♂️ Edital Dinâmico PC-AP - Elite Edition

Sistema dinâmico de gestão de estudos para o concurso da **Polícia Civil do Amapá**. 
Desenvolvido para organizar o progresso em matérias como Direito, Informática e História/Geografia do Amapá.

### 🚀 Funcionalidades
* **Autenticação Segura:** Login via Firebase para salvar seu progresso na nuvem.
* **Controle de Ciclos:** Gestão de Teoria, Questões e Revisões (7 e 28 dias).
* **Dashboard:** Gráficos interativos com seu domínio por disciplina.
